阿里云服务器Ubuntu16.04-PHP开发环境安装
发表于 2020-02-07| 更新于 2020-02-07|环境搭建
阅读量:

刚租了个学生服务器，半年免费，之前都是弄虚拟机，整个玩玩吧
打算先搭个LAMP（Linux- Apache-MySQL-PHP）环境

结果报错了

Code

原来是没有更新APT库，执行代码更新一下

Code
Code
  1. 安装Apache：
    ~# apt-get install apache2

But 安装成功后浏览器却不能访问http://IP

本机ping服务器也能ping通服务器，修改安全组规则：

现在可以访问了

  1. 安装数据库：
    # apt-get install mysql-server mysql-client

  2. 安装php
    # apt-get install php7.0

4.安装其他模块
apt-get install libapache2-mod-php7.0
apt-get install php7.0-mysql
顺便重启apache2和mysql服务：
service apache2 restart
service mysql restart

测试一下：在/var/www/html目录下编辑php文件，然后浏览器访问http://IP/1.php

<?php echo "hello yogurt!"; ?>

  1. 安装phpadmin
    apt-get install phpmyadmin

创建phpMyAdmin快捷方式：ln -s /usr/share/phpmyadmin /var/www/html

启用Apache mod_rewrite模块：a2enmod rewrite

重启服务：
service php7.0-fpm restart
service apache2 restart

访问http://ip/phpmyadmin 登陆管理页面

  1. 配置apache

/etc/apache2/apache2.conf中添加这句话：
AddType application/x-httpd-php .php .htm .html
AddDefaultCharset UTF-8

然后重启apache服务
