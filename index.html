中间件的定义

中间件（Middleware）是提供系统软件和应用软件之间连接的软件，以便于软件各部件之间的沟通。中间件处在操作系统和更高一级应用程序之间。他充当的功能是：将应用程序运行环境与操作系统隔离，从而实现应用程序开发者不必为更多系统问题忧虑，而直接关注该应用程序在解决问题上的能力 。容器就是中间件的一种。

CTFd的搭建

2019-04-13 Comments CTF环境搭建

本次搭建是基于ubuntu16.04。

1.升级源
sudo apt-get update

2.安装git

因为ctfd的源码和部署好的题目都要通过github传输

sudo apt install git

3.安装pip　
安装的时候可能会有点慢，可以更改一下源

sudo apt install python-pip

4.安装Flask

CTFd是基于Flask框架建造的

sudo pip install Flask

5.下载CTFd

sudo git clone https://github.com/isislab/CTFd.git

6.安装CTFd

cd CTFd

sudo ./prepare.sh

7.运行CTFd

sudo python serve.py

不出意外现在可以用本地访问了，但是我们搭建的ctfd不都是要通过外网让别人访问么。所以，还需要进行操作。

8.安装gunicorn并规定映射的端口
sudo pip install gunicorn

sudo gunicorn --bind 0.0.0.0:8000 -w 1 "CTFd:create_app()"

这里我定的是8000，端口可以自己随便改，那么现在我就可以通过ip+端口来访问ctfd了。不过现在的ctfd是最基础的页面，还需要进一步的调试。

这个命令要在CTFd的目录下执行

但是哇，这样太慢。所以打算弄个nginx+uwsgi的。提高性能。

执行安装nginx时总是失败，报错：
E: Sub-process /usr/bin/dpkg returned an error code (1)
然后解决措施：
进入目录：cd /var/lib/dpkg
删除文件：sudo mv info info.baksudo
重新创建：sudo mkdir info

然后我安装uwsgi时也报错了！

然后设置超时时间同时下载：pip –default-timeout=100 install uwsgi

然后又报错！

蓝后我在命令前加了sudo。
sudo pip --default-timeout=100 install uwsgi
成功下载。

Linux——文件的权限

2019-03-19 Comments Linux

文档的权限 UGO

访问文件是需要权限的，要有区分。这里记录的是UGO。
什么是UGO?

一般权限有三种对象：
属主：U
属组：G
其他人：O
基本权限类型：
读：r 4
写：w 2
执行：x 1
