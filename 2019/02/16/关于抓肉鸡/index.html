关于抓肉鸡

2019-02-16 Comments 随笔

‘抓鸡’抓的是可以背黑客控制的计算机，”肉鸡”可以是各种系统,如2003,2000,2008,xp,win7,win8,linux,unix等；更可以是一家公司\企业\学校甚至是政府军队的服务器。一旦成为了肉鸡之后黑客们就可以像操作自己的电脑那样来操作它而不被对方所发觉。
黑客们经常用80,8080,135,445,1433,1521,3306,3389 ,4899,5900之类的端口来抓鸡。现在许多人把有WEBSHELL 权限的机器也叫肉鸡。
黑客们利用工具就能抓到肉鸡。
https://www.cnblogs.com/heikeboke/p/8043415.html
这篇文章解释了3389“肉鸡”和“4489肉鸡”。

不过抓鸡是犯法的奥，如果是内部测试玩玩的话，还是可以的。

抓鸡，常见的就是利用系统弱口令进行暴力破解，从而拿到系统权限的一个过程。新手就可以玩，不需要高深技术水平。

通过扫描多个网段开放的下面这些端口，使用暴力破解软件进行破解:

  1. 远程桌面服务，默认3389端口
  2. MYSQL数据库，默认3306端口
  3. MSSQL数据库，默认1433端口
  4. FTP服务，默认端口21
  5. SSH服务，默认端口22
    and so on………………

首先呢要扫描ip，例如你想扫开启3306端口的主机，然后会扫到一批开启了3306端口的主机IP地址。扫ip的工具很多。网上都可以下载到。

然后将ip导入爆破工具里，导入用户字典和密码字典，从而成功登陆远程主机。DUbrute工具也是其一，这种爆破工具很多，功能也很多，掌握几个玩玩就可以了。

获得权限以后可以上传自己制作的木马或者其他操作。
