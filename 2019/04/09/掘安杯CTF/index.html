该死的温柔

可以发现该死和guess谐音，有个隐写工具OutGuess，它是一种通用的隐写工具，可以插入将隐藏信息存入数据源的冗余位
Kail终端命令输入：
git clone https://github.com/crorvick/outguess
安装包随即下载到文件夹。双击打开文件夹，右键点击空白区域终端打开。

随后输入以下命令 进行安装
./configure&& make && make install
右键图片查看属性，发现备注有个guess

这就是要工具解密的密码

然后构造命令：
root@kali:~/outguess# outguess -k "guess" -r '/root/桌面/flag.jpg' ctf.txt

其中guess是密码，-r是解密指令，ctf.txt是输出ctf.txt文件

打开ctf.txt文件，就可以显示flag了
