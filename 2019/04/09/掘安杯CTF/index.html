贝斯家族三英战群魔！

给了个文本：

贝斯家族三英让我们想起了base16，base32，base64.
于是将该文本base16，base32，base64依次解密四次。得到结果。
解密方法可以到解密网站解密，也可以脚本解密。
脚本如下：

Code

jactf{4(b64_32_16)}

罗马帝国的奠基者

一看题目就觉得是凯撒密码有关，百度一搜果然是凯撒。 这题300分白瞎了。
题目叙述为：

Code

仔细观察字符，h ^ _ o ` 的ASCII码分别为 104、94、95、111、96
再看j a c t f 的ASCII码分别为 106、97、99、116、102
从第一个字符开始，相差2，第二个字符相差3，后面依次4、5…… 于是脚本吧。

Code

jactf{bxcsium}

该死的温柔

可以发现该死和guess谐音，有个隐写工具OutGuess，它是一种通用的隐写工具，可以插入将隐藏信息存入数据源的冗余位
Kail终端命令输入：
git clone https://github.com/crorvick/outguess
安装包随即下载到文件夹。双击打开文件夹，右键点击空白区域终端打开。

随后输入以下命令 进行安装
./configure&& make && make install
右键图片查看属性，发现备注有个guess

这就是要工具解密的密码

然后构造命令：
root@kali:~/outguess# outguess -k "guess" -r '/root/桌面/flag.jpg' ctf.txt

其中guess是密码，-r是解密指令，ctf.txt是输出ctf.txt文件

打开ctf.txt文件，就可以显示flag了

你对我的网站做了什么？

分析流量包，打开shell.pcap。 查看HTTP流量包。
在一个post请求中发现了catflag。

再往下 在它的返回包看见一段加密字符。

于是追踪一下tcp流发现一段加密代码

于是写代码 反过来解密。得到flag。

这是什么玩意儿

简单，见过的一眼就能看出来。
先是一堆数字字母跟=的组合，是Quoted-Printable编码。解密网址
http://www.mxcz.net/tools/quotedprintable.aspx

得到了佛语，所以进行与佛论禅解密。
http://www.keyfc.net/bbs/tools/tudoucode.aspx

然后得到了社会主义核心价值观编码，接着解密。
http://www.nicetool.net/app/core_values.html


得到flag。
